# Changelog

All notable changes to `filament-diffs` will be documented in this file.

## Unreleased

- `DiffEntry` infolist component with `old()` / `new()` closure-based API

- Publishable config file for application-wide defaults
- Syntax highlighting via [Shiki](https://shiki.style/) (all languages and themes supported)
